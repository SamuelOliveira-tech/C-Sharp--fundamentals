# Exercício de Fixação - String Formatting em C#

**Objetivo:**  
Praticar a formatação de strings, números e caracteres em C#, incluindo Interpolação, Concatenação, Place Holder, arredondamento e uso de 'InvariantCulture'.

## Variáveis usadas

C#( Csharp )
string produto1 = "Computador";
string produto2 = "Mesa de escritório";

byte idade = 30;
int codigo = 5290;
char genero = 'M';

double preco1 = 2100.0;
double preco2 = 650.50;
double medida = 53.234567;

## Saida esperada

Produtos:
Computador, cujo preço é $ 2100,00
Mesa de escritório, cujo preço é $ 650,50

Registro: 30 anos de idade, código 5290 e gênero: M

Medida com oito casas decimais: 53,23456700
Arredondado (três casas decimais): 53,235
Separador decimal invariant culture: 53.235
